사용법.
우선 경로를 찾아가서 (...\adjust , ...\bus , ...\subway) gcc -o 원하는exe파일명 모든c파일, 모든h파일 ( ex) adjust일 경우, gcc -o adjust getinfo.c getinfo.h )
만들어진 각 exe파일 (총 3개)을 하나의 폴더안에 넣는다. ./파일명 으로 실행시킨다.

구조.
1. 모든 파일에서 시간을 재고, 정산되었음을 보여준다. (언뜻 편해보이나 불편한 것이 더 많음)
2. 기존에 있지 않은 ID를 입력할 때, 아이디를 새로 생성해주고 초기 충전값을 물어본다.(충전은 최초 한번만 가능)
3. 요금이 부족할 때, 일단 승차 하차 여부를 물어본 뒤 역까지 선택한 후에 부족하다는 메시지를 띄어준다.(요금 부족을 걸러주는게 뒤에 있나봄.)
4. 정산시스템이 제대로 되어있지않은것 같다.(내가 잘 모르는 걸수도.. 허허)


오류.
1. 버스 승차후 하차단말기를 태그하지 않고 지하철 승차했는데 기본요금이 나옴
2. 지하철에서 a라는 이름을 만들었으나버스에서 a를 치니 새로 만듬
3. core dumped가 일어남(이유를 잘 모르겠음)


평가.
1.우선 Thread인진 모르겠으나 시간이 계속 입력되어 입력을 방해하여서 불편 (Backspace가 소용이 없음)
2. display 해주는 info가 제대로 정리되어 있지 않은 것 같다.
3. 지하철을 하나의 exe로 통합하여 창이 많지않은 점은 편리하다.
4. core dumped로 인해 더이상의 평가가 불가능해짐
